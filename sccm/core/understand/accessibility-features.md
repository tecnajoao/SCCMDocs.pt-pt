# <a name="accessibility-features-in-system-center-configuration-manager"></a>Функции специальных возможностей в System Center Configuration Manager

*Применимо к: System Center Configuration Manager (Current Branch)*


В System Center Configuration Manager имеются функции, облегчающие работу людям с ограниченными возможностями.


## <a name="bkmk_aconsole"></a> Функции специальных возможностей для консоли Configuration Manager  

**Сочетания клавиш и улучшения начиная с версии 1706**

|Сочетание клавиш|  Назначение|
|--------|--------|  
|Ctrl+M|Устанавливает фокус на главной (центральной) панели.|
|Ctrl+T|Устанавливает фокус на верхнем узле на панели навигации. Если фокус уже находится в этой области, он перемещается к последнему посещенному узлу.|
|Ctrl+I|Устанавливает фокус на панели строки навигации, которая расположена под лентой.|
|Ctrl+L|Устанавливает фокус на поле **поиска**, если оно доступно.|
|Ctrl+D|Устанавливает фокус на области сведений, если она доступно.|
|Alt     |Переносит фокус на ленту или убирает фокус с ленты.|


- Улучшение навигации в области навигации при вводе букв имени узла.
- Теперь между основным представлением и лентой выполняется циклическая навигация.
- Теперь в области сведений выполняется циклическая навигация с помощью клавиатуры. Чтобы вернуться к предыдущему объекту или панели, нажмите клавиши CTRL+D, а затем SHIFT+TAB.
- Когда представление рабочей области обновится, фокус будет установлен на главной панели этой рабочей области.
- Устранена проблема, мешавшая средствам чтения с экрана сообщать имена элементов списка.
- Добавлены описательные имена для нескольких элементов управления на странице, позволяя средствам чтения с экрана сообщать важные сведения.


**Доступны следующие сочетания клавиш для всех версий**

- Для доступа к рабочим областям используйте следующие сочетания клавиш.  

|Сочетание клавиш| Рабочая область|
|--------|--------|  
|CTRL+1| Активы и соответствие|
|CTRL+2|  Библиотека программного обеспечения|
|CTRL+3|  Мониторинг|
|CTRL+4|  Администрирование|


-   Для доступа к меню рабочей области нажимайте клавишу TAB, пока фокус не окажется на значке "Развернуть или свернуть". Затем нажмите клавишу со стрелкой вниз, чтобы открыть меню рабочей области.  

-   Для перемещения по меню рабочей области используйте клавиши со стрелками.  

-   Для доступа к различным разделам рабочей области используйте клавишу TAB и сочетание клавиш SHIFT+﻿TAB. Для перемещения в пределах раздела рабочей области, например ленты, используйте клавиши со стрелками.  

-   Для доступа к адресной строке, если фокус ввода находится на узле дерева, нажмите клавиши SHIFT+TAB три раза.  

-   На странице свойств или мастера можно перемещаться между полями с помощью сочетаний клавиш. Чтобы выбрать определенное поле, нажмите клавишу ALT в сочетании с соответствующим подчеркнутым символом (ALT+_).     

-  Для перехода между различными узлами рабочей области введите первую букву имени узла. При каждом нажатии клавиши курсор перемещается к следующему узлу, имя которого начинается с этой буквы. Если используется средство чтения с экрана, оно считывает имя этого узла.

> [!NOTE]  
>  Сведения, представленные в этом разделе, предназначены только для пользователей, которые приобрели лицензии на продукты корпорации Майкрософт в США. Если продукт приобретен за пределами США, можно воспользоваться информационной карточкой подразделения, которая входит в комплект поставки пакета программного обеспечения, или найти контактные данные служб технической поддержки Майкрософт на [веб-сайте специальных возможностей корпорации Майкрософт](http://go.microsoft.com/fwlink/?LinkId=8431). Чтобы узнать, доступны ли в вашем регионе продукты и услуги, описанные в этом разделе, свяжитесь с местным подразделением. Сведения о специальных возможностях доступны на других языках, включая японский и французский.  

##  <a name="bkmk_ahelp"></a> Функции специальных возможностей в справке Configuration Manager  
 Справка Configuration Manager содержит функции, благодаря которым она становится доступной для более широкого круга пользователей, включая пользователей с ограниченной подвижностью, слабым зрением и другими ограничениями.  

|Задача|Использование сочетания клавиш|  
|----------------|--------------------------------|  
|Отображение окна справки.|F1|  
|Перемещение курсора между панелями разделов и навигации справки (такими вкладками, как **Содержание**, **Поиск**и **Указатель**).|F6|  
|Смена активных вкладок (например, **Содержание**, **Поиск** и **Указатель**) на панели навигации.|ALT+подчеркнутая буква в названии вкладки|  
|Выбор следующего скрытого текста или следующей гиперссылки.|Вкладка|  
|Выбор предыдущего скрытого текста или предыдущей гиперссылки.|SHIFT+TAB|  
|Выполнение действия, связанного с выделенной командой "Показать все", "Скрыть все", со скрытым текстом или гиперссылкой.|Ввод|  
|Отображение меню **Параметры** для доступа ко всем командам на панели инструментов справки.|ALT+O|  
|Скрытие или отображение панели, которая содержит вкладки **Содержание**, **Поиск** и **Указатель**.|ALT+O, а затем нажмите клавишу T|  
|Отображение ранее просмотренного раздела.|ALT+O, а затем нажмите клавишу B|  
|Отображение следующего раздела в последовательности ранее просмотренных разделов.|ALT+O, а затем нажмите клавишу F|  
|Возврат на указанную домашнюю страницу.|ALT+O, а затем нажмите клавишу H|  
|Прекращение открытия раздела справки в окне справки (например, при необходимости остановить скачивание веб-страницы).|ALT+O, а затем нажмите клавишу S|  
|Открытие в браузере Windows Internet Explorer диалогового окна **Свойства обозревателя**, позволяющего изменить параметры специальных возможностей.|ALT+O, а затем нажмите клавишу I|  
|Обновление раздела, например связанной веб-страницы.|ALT+O, а затем нажмите клавишу R|  
|Печать всех разделов в книге или только выбранного раздела.|ALT+O, а затем нажмите клавишу P|  
|Закрытие окна справки.|ALT+F4|  

#### <a name="to-change-the-appearance-of-a-help-topic"></a>Изменение внешнего вида раздела справки  

1.  Чтобы настроить цвета, начертания и размеры шрифтов, которые используются в справке, откройте окно справки.  

2.  Откройте меню **Параметры** и выберите пункт **Свойства браузера**.  

3.  На вкладке **Общие** нажмите кнопку **Специальные возможности**. Выберите параметры **Не учитывать цвета, указанные на веб-страницах**, **Не учитывать шрифты, указанные на веб-страницах** и **Не учитывать размеры шрифтов, указанные на веб-страницах**. Можно также выбрать использование параметров, указанных в собственной таблице стилей.  

#### <a name="to-change-the-color-of-the-background-or-text-in-help"></a>Изменение цвета фона или текста в справке  

1.  Откройте окно справки.  

2.  Откройте меню **Параметры** и выберите пункт **Свойства браузера**.  

3.  На вкладке **Общие** нажмите кнопку **Специальные возможности**. Затем выберите параметр **Не учитывать цвета, указанные на веб-страницах**. Можно также выбрать использование параметров, указанных в собственной таблице стилей.  

4.  Чтобы настроить цвета, используемые в справке, на вкладке **Общие** нажмите кнопку **Цвета**. Снимите флажок **Использовать цвета Windows** и выберите нужные цвета шрифта и фона.  

    > [!NOTE]  
    >  В результате изменения цвета фона в разделах справки также изменится цвета фона веб-страниц в окне Windows Internet Explorer.  

#### <a name="to-change-the-font-in-help"></a>Изменение шрифта справки  

1.  Откройте окно справки.  

2.  Откройте меню **Параметры** и выберите пункт **Свойства браузера**.  

3.  На вкладке **Общие** нажмите кнопку **Специальные возможности**. Чтобы использовать те же параметры, которые используются в вашем экземпляре Windows Internet Explorer, установите флажки **Не учитывать шрифты, указанные на веб-страницах** и **Не учитывать размеры шрифтов, указанные на веб-страницах**. Можно также выбрать использование параметров, указанных в собственной таблице стилей.  

4.  Чтобы настроить начертание шрифта в справке, нажмите кнопку **Шрифты** на вкладке **Общие** и выберите нужное начертание.  

    > [!NOTE]  
    >  В результате изменения шрифта в разделах справки также изменится цвет фона веб-страниц в окне Windows Internet Explorer.
