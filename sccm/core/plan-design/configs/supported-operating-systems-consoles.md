# <a name="supported-operating-systems-for-system-center-configuration-manager-consoles"></a>Sistemas operativos suportados para as consolas do System Center Configuration Manager

*Aplica-se a: O System Center Configuration Manager (ramo atual)*


 Para suportar a consola do System Center Configuration Manager, os seguintes sistemas operativos necessitam de uma versão mínima do .NET Framework do 4.5.2. A exceção é o Windows 10, que requer um mínimo de .NET Framework 4.6.  

-   **Windows Server 2016**: Standard, Datacenter  
  - Windows Server 2016 é suportado a partir do Configuration Manager versão 1606, com o rollup de correção de KB3186654 (ou a versão de linha de base do 1606, que foi lançada em Outubro de 2016).  


-   **Windows Server 2012 R2** (x64): Standard, Datacenter  

-   **Windows Server 2012** (x64): Standard, Datacenter  

-   **Windows Server 2008 R2 com SP1** (x64): Standard, Enterprise e Datacenter  

-   **Windows 10** (x86, x64): Pro, Enterprise  

-   **Windows 8.1** (x86, x64): Professional, Enterprise  

-   **Windows 8** (x86, x64): Professional, Enterprise  

-   **Windows 7 com SP1** (x86, x64): Professional, Enterprise, Ultimate
