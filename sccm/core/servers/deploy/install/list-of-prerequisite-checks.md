# <a name="list-of-prerequisite-checks-for-system-center-configuration-manager"></a>Список проверок готовности для System Center Configuration Manager

*Применимо к: System Center Configuration Manager (Current Branch)*

Следующие разделы подробно описывают доступные проверки готовности.

Дополнительные сведения об использовании средства проверки готовности см. в статье [Средство проверки готовности к установке](prerequisite-checker.md).  

##  <a name="BKMK_Security"></a> Проверки готовности для прав доступа  
Ниже приводится список проверок, которые средство проверки готовности к установке выполняет для прав доступа.

|Выполненная проверка|Пояснение|Уровень серьезности|Применимость сайта|
|---|---|---|---|
|**Права администратора на сайте центра администрирования**|Проверяет, имеет ли учетная запись пользователя, запускающая программу установки Configuration Manager, права **администратора** на компьютере сайта центра администрирования. |Ошибка|Первичный сайт|
|**Права администратора на расширенном первичном сайте**|Проверяет, имеет ли учетная запись пользователя, запускающая программу установки, права **администратора** на автономном первичном сайте, который будет расширен.|Ошибка|Сайт центра администрирования|
|**Права администратора в системе сайта**|Проверяет, имеет ли учетная запись пользователя, запускающая программу установки Configuration Manager, права **администратора** на компьютере сервера сайта. |Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Права администратора компьютера сайта центра администрирования на расширенном первичном сайте**|Проверяет, имеет ли учетная запись компьютера сайта центра администрирования права **администратора** на автономном первичном сайте, который будет расширен.|Ошибка|Сайт центра администрирования|
|**Подключение к SQL Server на сайте центра администрирования**|Проверяет, имеет ли учетная запись пользователя, запускающая программу установки Configuration Manager на первичном сайте для присоединения к существующей иерархии, роль **sysadmin** в экземпляре SQL Server для сайта центра администрирования.|Ошибка|Первичный сайт|
|**Права администратора учетной записи компьютера сервера сайта**|Проверяет, имеет ли учетная запись компьютера сервера сайта права **администратора** на компьютерах SQL Server и точки управления.|Ошибка|Первичный сайт <br>SQL Server|
|**Связь системы сайта и SQL Server**| Проверяет, правильное ли имя субъекта-службы зарегистрировано в доменных службах Active Directory для учетной записи, настроенной для запуска службы SQL Server в экземпляре SQL Server, используемом для размещения базы данных сайта Configuration Manager. Для поддержки проверки подлинности Kerberos в доменных службах Active Directory должно быть зарегистрировано правильное имя субъекта-службы.|Предупреждение|Вторичный сайт <br>Точка управления.|
|**Режим безопасности SQL Server**|Проверяет, настроен ли SQL Server для обеспечения безопасности проверки подлинности Windows.|Предупреждение|SQL Server|
|**Права системного администратора SQL Server**|Проверяет, имеет ли учетная запись пользователя, которая запускает программу установки Configuration Manager, роль **sysadmin** в экземпляре SQL Server, выбранном для установки базы данных сайта. Эта проверка также завершается неудачно, если программе установки не удается получить доступ к экземпляру SQL Server для проверки разрешений.|Ошибка|SQL Server|
|**Права системного администратора SQL Server для эталонного сайта**|Проверяет, имеет ли учетная запись пользователя, которая запускает программу установки Configuration Manager, роль **sysadmin** в экземпляре SQL Server, выбранном в качестве базы данных эталонного сайта. Для изменения базы данных сайта требуются разрешения роли **sysadmin** SQL Server.|Ошибка|SQL Server|

##  <a name="BKMK_Dependencies"></a> Проверки готовности для зависимостей Configuration Manager
В следующей таблице приводится список проверок, которые средство проверки готовности к установке выполняет для зависимостей Configuration Manager.

|Выполненная проверка|Пояснение|Уровень серьезности|Применимость сайта|
|---|---|---|---|
|**Активные сопоставления миграции на целевой первичный сайт**|Проверяет, отсутствуют ли активные сопоставления миграции на первичные сайты.|Ошибка|Сайт центра администрирования|
|**Активная реплика пакета управления**|Проверяет наличие активной реплики точки управления.|Ошибка|Первичный сайт|
|**Права администратора в точке распространения**|Проверяет, имеет ли учетная запись пользователя, запускающая программу установки, права **администратора** на компьютере точки распространения.|Предупреждение|Точка распространения.|
|**Права администратора в точке управления**|Проверяет, имеет ли учетная запись компьютера сервера сайта права **администратора** на компьютере точки управления и точки распространения.|Предупреждение|Точка управления.|
|**Административный общий ресурс (в системе сайта)**|Проверяет, присутствуют ли необходимые административные общие ресурсы на компьютере системы сайта.|Предупреждение|Точка управления.|
|**Совместимость приложений**|Проверяет, соответствуют ли текущие приложения схеме приложений.|Предупреждение|Сайт центра администрирования <br>Первичный сайт|
|**Включена BITS**|Проверяет, установлена ли фоновая интеллектуальная служба передачи данных (BITS) на компьютере системы сайта точки управления. Если эта проверка завершается неудачно, BITS не установлена, на компьютере или удаленном хост-компьютере IIS не установлен компонент совместимости с WMI IIS 6.0 для IIS 7.0, или программе установки не удается проверить удаленные параметры IIS, так как на компьютере сервера сайта не установлены общие компоненты IIS.|Ошибка|Точка управления.|
|**Установлена BITS**|Проверяет, что служба BITS установлена в IIS.|Предупреждение|Точка управления.|
|**Сортировка без учета регистра в SQL Server**|Проверяет, использует ли установка SQL Server сортировку без учета регистра, такую как SQL_Latin1_General_CP1_CI_AS.|Ошибка|SQL Server|
|**Проверка версии и кода сайта существующего автономного первичного сайта**|Проверяет, является ли автономным первичный сайт, который планируется расширить, и имеет ли он ту же версию Configuration Manager, что и устанавливаемый сайт центра администрирования, но отличный от него код сайта.|Ошибка|Сайт центра администрирования <br>Первичный сайт|
|**Проверка несовместимых ссылок на коллекции**|Во время обновления эта проверка подтверждает, что коллекции ссылаются только на другие коллекции того же типа.|Ошибка|Сайт центра администрирования|  
|**Версия клиента на компьютере точки управления**|Проверяет, выполняется ли установка точки управления на компьютере, на котором отсутствует другая версия установленного клиента Configuration Manager.|Ошибка|Точка управления.|
|**Настройка использования памяти SQL Server**|Проверяет, настроен ли SQL Server для неограниченного использования памяти. Следует настроить максимальное предельное значение памяти SQL Server.|Предупреждение|SQL Server|
|**Выделенный экземпляр SQL Server**|Проверяет, настроен ли выделенный экземпляр SQL Server для размещения базы данных сайта Configuration Manager. Если этот экземпляр используется другим сайтом, необходимо выбрать другой экземпляр для использования новым сайтом. Либо можно удалить этот другой сайт или переместить его базу данных в другой экземпляр для SQL Server.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Существующие серверные компоненты Configuration Manager на сервере сайта**|Проверяет, не установлен ли сервер сайта или роль системы сайта на компьютере, выбранном для установки сайта.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Исключение брандмауэра для SQL Server**|Проверяет, не отключен ли брандмауэр Windows, или существует ли соответствующее исключение брандмауэра Windows для SQL Server. Необходимо разрешить удаленный доступ к sqlservr.exe или к требуемым TCP-портам. По умолчанию SQL Server прослушивает TCP-порт 1433, а служба SQL Server Service Broker (SSB) использует TCP-порт 4022.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт <br>Точка управления.|
|**Исключение брандмауэра для SQL Server (автономный первичный сайт)**|Проверяет, не отключен ли брандмауэр Windows, или существует ли соответствующее исключение брандмауэра Windows для SQL Server. Необходимо разрешить удаленный доступ к sqlservr.exe или к требуемым TCP-портам. По умолчанию SQL Server прослушивает TCP-порт 1433, а служба SSB использует TCP-порт 4022.|Предупреждение|Первичный сайт (только автономный)|
|**Исключение брандмауэра для SQL Server для точки управления**|Проверяет, не отключен ли брандмауэр Windows, или существует ли соответствующее исключение брандмауэра Windows для SQL Server.|Предупреждение|Точка управления.|
|**Конфигурация HTTPS IIS**|Проверяет привязки веб-сайта IIS для протокола сетевого взаимодействия HTTPS. Если выбирается установка ролей сайта, которым требуется HTTPS, то необходимо настроить привязки сайта IIS на конкретном сервере с использованием правильного PKI-сертификата.|Предупреждение|Точка управления <br>Точка распространения.|
|**Запущена служба IIS**|Проверяет, установлены ли службы IIS, и работают ли они на компьютере для установки точки управления или точки распространения.|Ошибка|Точка управления <br> Точка распространения.|
|**Соответствие параметров сортировки расширенного первичного сайта**|Проверяет, имеет ли база данных сайта автономного первичного сайта, который будет расширяться, те же параметры сортировки, что и база данных сайта на сайте центра администрирования.|Ошибка|Сайт центра администрирования|
|**Библиотека удаленного разностного сжатия Майкрософт (RDC) зарегистрирована**|Проверяет, зарегистрирована ли библиотека RDC на сервере сайта Configuration Manager.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Установщик Microsoft Windows**|Проверяет версию установщика Windows. Если эта проверка завершается неудачно, то либо программа установки не смогла проверить версию, либо установленная версия не соответствует минимальным требованиям установщика Windows версии 4.5.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Microsoft XML Core Services 6.0 (MSXML60)**|Проверяет, установлены ли на компьютере MSXML 6.0 или последующей версии.|Предупреждение|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт <br>Консоль Configuration Manager <br>Точка управления <br>Точка распространения.|
|**Минимальная версия .NET Framework для консоли Configuration Manager**|Проверяет, установлена ли на компьютере консоли Configuration Manager платформа Microsoft .NET Framework версии 4.0. Вы можете скачать NET Framework 4.0 в [Центре загрузки Майкрософт](http://go.microsoft.com/fwlink/p/?LinkId=189149).|Ошибка|Консоль Configuration Manager|
|**Минимальная версия .NET Framework для сервера сайта Configuration Manager**|Проверяет, установлена ли на компьютере консоли Configuration Manager платформа .NET Framework версии 3.5. Для Windows Server 2008 платформу Microsoft .NET Framework версии 3.5 можно скачать в [Центре загрузки Майкрософт](http://go.microsoft.com/fwlink/p/?LinkId=185604). Для Windows Server 2008 R2 можно включить .NET Framework версии 3.5 как компонент в диспетчере серверов.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Минимальная версия .NET Framework для установки SQL Server Express на вторичном сайте Configuration Manager**|Проверяет, что платформа .NET Framework 4.0 установлена на вторичных сайтах Configuration Manager, для установки SQL Server Express.|Ошибка|Вторичный сайт|
|**Параметры сортировки родительской и дочерней баз данных**|Проверяет, соответствуют ли параметры сортировки базы данных сайта параметрам сортировки базы данных родительского сайта. Все сайты в иерархии должны использовать одни и те же параметры сортировки базы данных.|Ошибка|Первичный сайт <br>Вторичный сайт|
|**PowerShell 2.0 на сервере сайта**|Проверяет, установлена ли среда Windows PowerShell 2.0 или более поздней версии на сервере сайта для коннектора Exchange для Configuration Manager. Дополнительные сведения о PowerShell 2.0 см. в [статье 968930](http://go.microsoft.com/fwlink/p/?LinkId=226450) базы знаний Майкрософт.|Предупреждение|Первичный сайт|
|**Первичное полное доменное имя**|С помощью полного доменного имени проверяет, соответствует ли NetBIOS-имя компьютера локальному имени узла (первой метке полного доменного имени) этого компьютера.|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт <br>SQL Server|
|**Удаленное подключение к инструментарию WMI на вторичном сайте**|Проверяет, может ли программа установки устанавливать удаленное подключение к инструментарию WMI на сервере вторичного сайта.|Предупреждение|Вторичный сайт|
|**Необходимые параметры сортировки SQL Server**|Проверяет, настроено ли использование параметров сортировки SQL_Latin1_General_CP1_CI_AS экземпляром SQL Server и базой данных сайта Configuration Manager (если он установлен), кроме случая, когда используется операционная система на китайском языке, и требуется поддержка GB18030.<br><br>Сведения об изменении параметров сортировки экземпляра и базы данных SQL Server см. в статье [Настройка и изменение параметров сортировки](http://go.microsoft.com/fwlink/p/?LinkID=234541) в электронной документации на SQL Server 2008 R2.  Сведения о включении поддержки GB18030 см. в статье [Международная поддержка в System Center Configuration Manager](../../../../core/plan-design/hierarchy/international-support.md).|Ошибка|Сайт центра администрирования <br>Первичный сайт <br>Вторичный сайт|
|**Папка исходных файлов установки**|Проверяет, имеет ли учетная запись компьютера для вторичного сайта разрешения на **чтение** файлов NTFS и общие разрешения на **чтение** в папке исходных файлов программы установки и в общем ресурсе.<br><br>**Примечание**. Если используются административные общие ресурсы (например, C$ и D$), то учетная запись компьютера вторичного сайта должна быть **администратором** на компьютере.|Ошибка|Вторичный сайт|
|**Версия исходных файлов установки**|Проверяет, соответствует ли версия Configuration Manager в папке исходных файлов, указанной для установки вторичного сайта, версии Configuration Manager первичного сайта.|Ошибка|Вторичный сайт|
|**Код сайта используется**|Проверяет, не используется ли указанный код сайта в иерархии Configuration Manager. Следует указывать уникальный код сайта для этого сайта.|Ошибка|Первичный сайт|
|**Компьютер поставщика SMS имеет тот же домен, что и сервер сайта**|Проверяет, не находится ли компьютер, на котором работает экземпляр поставщика SMS, в одном домене с сервером сайта.|Ошибка|поставщика SMS|
|**Выпуск SQL Server**|Проверяет, что выпуск SQL Server на сайте не является выпуском SQL Server Express.|Ошибка|SQL Server|
|**SQL Server Express на вторичном сайте**|Проверяет, можно ли успешно установить SQL Server Express на компьютере сервера сайта для вторичного сайта.|Ошибка|Вторичный сайт|
|**SQL Server на компьютере вторичного сайта**|Проверяет, что SQL Server установлен на компьютере вторичного сайта. Установка SQL Server на удаленной системе сайта не поддерживается.<br><br>**Предупреждение**. Эта проверка проводится только в том случае, если программа установки использует существующий экземпляр SQL Server.|Ошибка|Вторичный сайт|
|**Распределение памяти процесса SQL Server**|Проверяет, что SQL Server резервирует не менее 8 ГБ памяти для сайта центра администрирования и первичного сайта, и не менее 4 ГБ памяти для вторичного сайта. Дополнительные сведения о выделении фиксированного объема памяти с помощью SQL Server Management Studio см. в статье [Практическое руководство. Установка фиксированного размера памяти (среда SQL Server Management Studio)](http://go.microsoft.com/fwlink/p/?LinkId=233759).<br><br>**Примечание**. Данная проверка неприменима к экспресс-выпуску SQL Server на вторичном сайта, который ограничен 1 ГБ зарезервированной памяти.|Предупреждение|SQL Server|
|**Учетная запись запуска службы SQL Server**|Проверяет, что учетная запись входа для службы SQL Server не является учетной записью локального пользователя или LOCAL SERVICE.
