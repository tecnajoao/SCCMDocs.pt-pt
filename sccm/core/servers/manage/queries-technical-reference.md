# <a name="queries-technical-reference-for-system-center-configuration-manager"></a>Referência técnica para consultas no System Center Configuration Manager

*Aplica-se a: O System Center Configuration Manager (ramo atual)*

Consultas no System Center Configuration Manager devolvem informações da base de dados do site com base nos critérios que especificar. Pode utilizar consultas para obter informações sobre os recursos no site ou os dados do inventário e as mensagens de estado.
